ðŸ”¹ SQL Joins

INNER JOIN

Returns only matching rows from both tables.

Most commonly used.

LEFT JOIN (or LEFT OUTER JOIN)

Returns all rows from the left table and matching rows from the right.

If no match, NULL is returned from the right table.

FULL JOIN (or FULL OUTER JOIN)

Returns all rows from both tables, matched or unmatched.

If no match, NULL is shown for missing values.

SELF JOIN

Join a table with itself.

Used for hierarchical data (example: employee & manager relationship).

SELECT e.emp_name, m.emp_name AS manager
FROM employees e
JOIN employees m ON e.manager_id = m.emp_id;

Returns all combinations â†’ Formula:

Total Rows = Rows(Table A) Ã— Rows(Table B)


Example: 5 rows Ã— 4 rows = 20 rows.

Used for testing and sometimes in Python pandas.
