ðŸ”¹ 1. Select column values

To take data from specific column â†’

SELECT emp_name FROM emp;


This will return only the emp_name column (not all columns).

ðŸ”¹ 2. Row-wise filtering (WHERE clause)

Filter specific rows using WHERE:

SELECT emp_name FROM emp WHERE emp_id = 2;


ðŸ‘‰ Only employee with emp_id = 2 will be shown.

ðŸ”¹ 3. LIKE operator

Used for pattern matching.

Starts with "R":

SELECT emp_name FROM emp WHERE emp_name LIKE 'R%';


Ends with "A":

SELECT emp_name FROM emp WHERE emp_name LIKE '%A';


Contains "a" anywhere:

SELECT emp_name FROM emp WHERE emp_name LIKE '%a%';

ðŸ”¹ 4. Aggregate Functions

Aggregate functions = summarize multiple rows.

Count total rows

SELECT COUNT(*) FROM emp;


Maximum salary

SELECT MAX(salary) FROM emp;


Minimum salary

SELECT MIN(salary) FROM emp;


Average salary

SELECT AVG(salary) FROM emp;


Sum of salary

SELECT SUM(salary) FROM emp;

ðŸ”¹ 5. GROUP BY

Used to group rows based on a column â†’ useful in reports/dashboard.

Example: Average salary by department:

SELECT dept_id, AVG(salary) 
FROM emp 
GROUP BY dept_id;


ðŸ‘‰ Without GROUP BY â†’ only one average for whole table.
ðŸ‘‰ With GROUP BY â†’ one average for each department.

ðŸ”¹ 6. HAVING

WHERE = filter individual rows.

HAVING = filter grouped results.

Example: Only show departments with average salary > 30000:

SELECT dept_id, AVG(salary) 
FROM emp 
GROUP BY dept_id 
HAVING AVG(salary) > 30000;

ðŸ”¹ 7. LIMIT

Used to restrict number of rows in result (MySQL, PostgreSQL).

First 2 rows:

SELECT * FROM emp LIMIT 2;


First 5 employees with highest salary:

SELECT * FROM emp ORDER BY salary DESC LIMIT 5;

ðŸ”¹ 8. Extra (for dashboards / audit columns)

Many companies keep tracking columns in tables:

inserted_by â†’ who inserted the record

inserted_at â†’ when it was inserted

updated_by â†’ who last updated

updated_at â†’ when it was updated

ðŸ‘‰ Example table:

CREATE TABLE emp (
  emp_id INT PRIMARY KEY,
  emp_name VARCHAR(50),
  salary DECIMAL(10,2),
  dept_id INT,
  inserted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  inserted_by VARCHAR(50),
  updated_at TIMESTAMP,
  updated_by VARCHAR(50)
);
